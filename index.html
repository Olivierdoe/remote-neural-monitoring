Remote Neuromodulation is a technology that allows generating sensory experiences by wirelessly altering brain waves. It allows anyone to be engaged in a full immersive virtual reality. The most common ways to achieve this are by delivering specifically engineered genes that code for a magnetised protein to the brain using an AAV virus or by delivering neuralnanorobots to the brain parenchyma and then altering brain waves using electromagnetic waves.
